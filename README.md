# city-timezone-vercel

Минимальный проект для Vercel:

- открытая HTML-страница
- поиск населённого пункта через Google Places API (New)
- получение координат выбранного населённого пункта
- получение часового пояса на выбранную дату через Google Time Zone API
- место для вызова второго API и интерпретации его ответа

## Структура

- `index.html` — публичная страница
- `api/autocomplete.js` — подсказки населённых пунктов
- `api/city-info.js` — координаты, timezone и второй API
- `.env.example` — пример переменных окружения

## Как запустить локально

1. Установить Vercel CLI:
   `npm i -g vercel`
2. В корне проекта выполнить:
   `vercel dev`
3. Открыть адрес из консоли.

## Как задеплоить

1. Создать репозиторий и загрузить сюда файлы.
2. Импортировать репозиторий в Vercel.
3. В Settings -> Environment Variables добавить:
   - `GOOGLE_MAPS_API_KEY`
   - `OTHER_API_URL`
   - `OTHER_API_KEY`
4. Нажать Redeploy.
